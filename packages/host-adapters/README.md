# @runxhq/host-adapters

Thin host response adapters over the runx host protocol.

This package remains a published TypeScript bridge after the Rust takeover.
Its contract surface is the host protocol shape exported through
`@runxhq/contracts`; it does not own runtime execution.

See the [TypeScript interop boundary](../../docs/ts-interop-boundary.md) for
the package disposition and ownership rules.
